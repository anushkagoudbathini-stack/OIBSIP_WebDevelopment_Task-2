# Portfolio Website

## Objective

To design and develop a personal portfolio website using HTML and CSS that showcases skills, projects, and basic personal information in a structured and visually appealing manner.

## Steps

1. Planned the layout and sections of the portfolio website
2. Created the website structure using HTML5
3. Styled the website using CSS3 for layout and responsiveness
4. Tested the website across different devices and screen sizes

## Technologies Used

* HTML5
* CSS3

## Purpose

This project was developed to demonstrate frontend web development skills and present personal and project details through a professional portfolio website.
